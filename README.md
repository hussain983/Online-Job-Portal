# Online Job Portal System

## Overview

The **Online Job Portal System** is a full-stack web application developed to simplify the recruitment process by connecting job seekers with employers on a single platform. The system provides an efficient and user-friendly environment where employers can post job vacancies, manage applications, and review candidate profiles, while job seekers can search for jobs, submit applications, and manage their personal information.

The application is built using **HTML5**, **CSS3**, **JavaScript**, **PHP**, and **MySQL**, following a structured client-server architecture. The frontend offers a responsive and intuitive interface, while the backend handles business logic, authentication, database operations, and user management.

## Features

### Employer Module

* Employer registration and secure login.
* Create, update, and delete job postings.
* View and manage applications submitted by job seekers.
* Manage employer profile and company information.

### Job Seeker Module

* User registration and authentication.
* Search and filter job opportunities.
* Apply for available jobs online.
* Manage personal profile and application history.

### General Features

* Secure user authentication using PHP sessions.
* Role-based access for employers and job seekers.
* CRUD operations for users, jobs, and applications.
* Responsive design for desktop and mobile devices.
* Structured relational database using MySQL.
* Input validation and form handling for improved reliability.

## Technologies Used

* **Frontend:** HTML5, CSS3, JavaScript
* **Backend:** PHP
* **Database:** MySQL
* **Server Environment:** XAMPP (Apache & MySQL)
* **Database Management:** phpMyAdmin
